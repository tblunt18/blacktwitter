The idea of free expression on social media is extremely pivotal particularly for African-Americans whose voices are often silenced. A popular platform is Black Twitter. Given the results of the past presidential election, conversation has sparked across all media platforms. The 2017 Academy Awards provide an opportunity to gather and analyze a large number of tweets that would be potentially political and or controversial. Using the Jupyter notebook platform, we can better visualize the data leveraging various API libraries exist such as pandas, plot.ly, json and Django framework. Plot.ly allows us to compare the highly used hashtags and keywords for the Oscars 2016 and 2017 through two separate interactive U.S. maps.  These geographical locations provide insights into how a user’s location can correlate to the social context of their tweets 

The Oscars awards is often a controversial event year after year. The show is laced with political irony and underlying issues existing in the U.S., and we collected interesting data to visualize

  • 2016
  • 2017
  • 2018
For the 2016 Oscars awards, our focus was to gather twitter data using Twitter's Tweepy API and Python. We streamed tweets during the awards based on current hashtags such as #OscarsSoWhite
In 2017, it was time to collect more tweets and formulate a comparison between how those existing within the realm of Black Twitter felt about the show in the previous year. We generated word clouds to show the comparison using the Python coding language.
This upcoming research day we would like to make our research interactive by providing a tool that students & faculty can use to generate their own clouds and to visualize the tweets
Our Playground is an interactive map that allows users to play around with the contents of tweets. It combines all of the programs that were written using Python and JavaScript to leverage functions for activitiy and analysis.

Features
-----------
- Create word clouds based on a collection of tweets
- Visualize what tweets came from specific locations within the U.S.
- Group words based on similarity and proximity (meaning)
- Steam tweets based on specific key words
- Save various clouds and streams for future use
